## ⚛️ React {#react}

🔐 [Mini Shai-Hulud worm keeps digging](https://www.wiz.io/blog/mini-shai-hulud-teampcp-hits-antv-supply-chain)

Remember last week’s TanStack Router compromise? The dangerous worm from TeamPCP keeps compromising users and maintainers across ecosystems, affecting popular packages such as `echarts-for-react` and `@antv`, and GitHub actions such as `actions-cool/issues-helper`. [OpenAI employees got compromised](https://openai.com/index/our-response-to-the-tanstack-npm-supply-chain-attack/), leading them to regenerate code signing certificates. The Million.js repository was also [briefly compromised](https://github.com/aidenybai/million/issues/1186), but they seem to have gotten lucky: it apparently didn’t affect anyone.

Rumors also link this worm to the [Nx Console VSCode extension compromise](https://github.com/nrwl/nx-console/security/advisories/GHSA-c9j4-9m59-847w), and possibly connected to a major [GitHub internal repo breach](https://x.com/H4ckmanac/status/2057032154601709611). This story isn’t over — “Mini” feels like a very questionable name 😅!

## 🔀 Other {#other}

- 👀 [Explicit Resource Management reached TC39 stage 4](https://github.com/tc39/proposal-explicit-resource-management) \- A TC39 meeting is in progress, and many proposals have progressed. The most “user-facing” one – `using` – is going to be part of the standard and simplify your resource cleanup logic. I take the opportunity to show you [how to leverage it immediately in Vitest/Jest tests](https://x.com/sebastienlorber/status/2057033638827483236).
- 👀 [npm RFC \- Make install scripts opt-in](https://github.com/npm/rfcs/pull/868) \- It’s great to see npm finally looking to catch up on security\! They’d like to introduce an `allowScripts` map in `package.json` to help mitigate the recent supply chain attacks.
- 👀 [Bun PR \- Rewrite Bun in Rust](https://github.com/oven-sh/bun/pull/30412) \- The bold, AI-driven PR landed in just one week, making Bun’s migration from Zig to Rust official. 🤯
- 📜 [Declarative partial updates](https://developer.chrome.com/blog/declarative-partial-updates) \- New exciting APIs may come to the web platform: out-of-order streaming, and new HTML insertion methods supporting streaming. Already available in Chrome 148 behind an experimental flag.
- 📜 [QUIC and HTTP/3 Come To Node.js (finally)](https://www.jasnell.me/posts/quic-comes-to-node) \- A 5-part deep dive on the experimental `node:quic` API.
- 📦 [pnpm 11.12 \- Experimental Rust install engine](https://github.com/pnpm/pnpm/releases/tag/v11.2.0) \- pnpm is also [migrating to Rust](https://github.com/pnpm/pnpm/tree/main/pacquet), with an incremental strategy. This release introduces an opt-in for the new Rust engine to power the install phase.
- 📦 [Yarn 4.15 \- Add `npmMinimalAgeGate: 1d` release cooldown by default](https://github.com/yarnpkg/berry/releases/tag/%40yarnpkg%2Fcli%2F4.15.0) \- Another package manager adopts better security defaults. BTW, Yarn is also being rewritten in Rust.
- 📦 [Node 26.2 \- `stream.compose` stable, `fs.stat()` returns `Temporal.Instant`](https://nodejs.org/en/blog/release/v26.2.0)
