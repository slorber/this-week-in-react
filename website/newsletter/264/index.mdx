[**Voltra 1.0 \- Live Activities & Widgets with JSX in React Native**](https://www.callstack.com/blog/live-activities-and-widgets-with-react-say-hello-to-voltra)

Voltra is a new library that allows React Native devs compose Live Activities and Widgets using SwiftUI primitives in just JSX, without having to write native code.

While previous community attempts, such as [expo-apple-targets](https://github.com/EvanBacon/expo-apple-targets) or [expo-live-activity](https://github.com/software-mansion-labs/expo-live-activity), either supported a limited set of possible layouts or required knowledge of Swift, Voltra seems to overcome these limitations.

Note that Expo is also working on their own [solution for Widgets & Live Activities](https://x.com/k7grzywacz/status/2000980494998175934) coming in Expo SDK 55\.
