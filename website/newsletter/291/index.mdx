- 💬 [React Native RFC \- Binary Props \- Phasing folly::dynamic out of Fabric's Android update pipeline](https://github.com/bartlomiejbloniarz/discussions-and-proposals/blob/binary-props/proposals/1013-binary-props-pipeline.md) \- When RN updates props on Android, it serializes every changed prop into a `folly::dynamic object`. The proposed fix will write prop diffs into a single shared binary. The result is \~257 µs → \~36 µs per view per frame on an entry-level phone. For 200 animated views: 64.3ms → 17.1ms total.
