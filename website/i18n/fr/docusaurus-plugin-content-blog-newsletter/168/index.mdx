## ⚛️ React {#react}

[![TanStack Router 1.0](/emails/issues/168/tanstack-router.jpg)](https://tanstack.com/router/v1)

**[TanStack Router 1.0](https://tanstack.com/router/v1)**

Le très attendu TanStack Router a été lancé en version stable 1.0 avec un joli [🐦 trailer](https://twitter.com/tannerlinsley/status/1738709130213560497), juste à temps pour Noël. Ce nouveau routeur repousse les limites en termes de typesafety, de gestion de l'état de l'url (search params) et de mise en cache, entre autres.

Le [guide SSR](https://tanstack.com/router/v1/docs/guide/ssr) fait référence à un package `@tanstack/react-start`, probablement un teaser pour un prochain framework [🐦 TanStack Start](https://twitter.com/sebastienlorber/status/1742589454563369123)

D'autres packages TanStack récents que vous auriez pu manquer :

- 📦 [TanStack Config - Tool for publishing and maintaining high-quality JavaScript packages](https://github.com/TanStack/config)
- 📦 [TanStack Store - Framework agnostic, type-safe store with reactive framework adapters](https://tanstack.com/store/latest)
