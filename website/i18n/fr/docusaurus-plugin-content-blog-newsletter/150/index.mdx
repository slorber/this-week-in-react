## ⚛️ React {#react}

[![](/emails/issues/150/nextjs-update.jpg)](https://nextjs.org/blog/june-2023-update)

**[Next.js App Router Update - Top Priorities](https://nextjs.org/blog/june-2023-update)**

L'équipe Next.js donne un peu de contexte sur le nouvel App Router. Ils ont reçu pas mal de critiques dernièrement et sont bien conscients des bugs et autres problèmes de DX actuelle. Ils détaillent ce sur quoi ils travaillent en priorité:

- La performance: local, build et server runtime
- Les bugs et instabilités
- L'éducation et la documentation

[![](/emails/issues/150/skew-protection.jpg)](https://vercel.com/blog/version-skew-protection)

**[Vercel - Introducing Skew Protection](https://vercel.com/blog/version-skew-protection)**

Nouvelle feature Vercel qui permet d'atténuer les désynchronisations back/front qui peuvent survenir lors d'un nouveau déploiement: les requêtes des anciens clients seront routées sur le déploiement immuable approprié. Particulièrement utile pour ceux qui font du déploiement continu. Disponible en expérimental dans Next.js 13.4.7 qui va automatiquement appliquer les bons headers HTTP.

**[Shadcn UI update - June 2023](https://ui.shadcn.com/docs/changelog)**

Cette collection de composants Tailwind + Radix fait de plus en plus d'adeptes. L'auteur vient de publier une grosse mise à jour avec un CLI qui propose déjà une belle DX, et semble avoir l'ambition d'y apporter prochainement des [features AI impressionnantes](https://twitter.com/shadcn/status/1673055193968287744). Kent C. Dodds [explique](https://twitter.com/kentcdodds/status/1673806183935909888) pourquoi il a adopté Shadcn UI dans sa stack Remix.
