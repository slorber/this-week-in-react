[Astro 3.0](https://astro.build/blog/astro-3/)

Sitepoint: Open source web framework Astro just reached its 3.0 milestone. This realease brings a ton of cool features, notably a significant performance bump (30-75% faster component render), and it also brings support for view transitions, enabling you to fade, slide, morph, and even persist stateful elements across page navigation with ease.
